We are implementing the Genus 0, simple radical extension case of the Risch-Trager algorithm in Mathematica. Refer to @TragerPlan.md for the implementation plan.

The code should be modular, clearly written, with extensive documentation and inline comments. 

Every function should have extensive unit tests and be tested in Mathematica.

Refer to Trager's thesis and other relevant papers by Miller, Kauers, Schults etc in @papers/ These should be carefully studied prior to writing any code. 

Implementations should follow the algorithms as described in Trager's 1984 thesis (@papers/Trager 1984 - Integration of algebraic functions.pdf). Inline comments should describe the underlying mathematics and cite the relevant Trager 1984 chapter / section; do not include code-level annotations referencing third-party implementations.

This project should NEVER use FullSimplify.
